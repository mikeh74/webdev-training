# Recipe Page

## Exercise Overview

Structure a recipe with headings, ingredients list, ordered steps, and an image.

## Objectives

- Create a well-structured recipe page using semantic HTML
- Use both unordered and ordered lists appropriately
- Include an appealing recipe image
- Organize content with clear headings and sections
- Make the content visually engaging

## Key HTML Elements to Use

- `<h1>`, `<h2>`, `<h3>` - Headings
- `<img>` - Recipe image
- `<ul>` - Unordered list for ingredients
- `<ol>` - Ordered list for cooking steps
- `<p>` - Descriptive paragraphs

## Suggested Structure

1. Main heading with recipe name
2. Brief description or introduction
3. An image of the finished dish
4. "Ingredients" section with an unordered list
5. "Instructions" section with an ordered list of steps
6. Optional: Prep time, cook time, servings information

## Tips

- Choose a recipe you like or create a fictional one
- Use descriptive alt text for the recipe image
- Consider adding sections for:
  - Preparation time
  - Cooking time
  - Servings/Yield
  - Difficulty level
- Use semantic HTML5 elements like `<article>` or `<section>` where appropriate
- Style with CSS to make the recipe easy to read and visually appealing
- Consider adding a print stylesheet for easy printing

## Example Recipe Ideas

- Chocolate chip cookies

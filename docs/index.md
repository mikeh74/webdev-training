# Web Development Training

Welcome to the Web Development Training documentation! This repository is designed as a comprehensive learning resource for aspiring web developers, covering everything from fundamental web technologies to building full-stack applications with Django.

## Purpose

This repository serves as a structured learning path for web development, providing:

- **Foundational Knowledge**: Understanding of HTML, CSS, and JavaScript - the core technologies of the web
- **Practical Exercises**: Hands-on coding challenges to build real-world skills
- **Framework Training**: Introduction to Django for building production-ready web applications
- **Career Guidance**: Overview of different roles and paths in web development

## Getting Started

If you're new to web development, we recommend starting with these foundational topics:

1. **[Job Roles in Web Development](job-roles.md)** - Understand the landscape of web development careers
2. **[Introduction to Web Development](intro-to-web.md)** - Learn about HTML, CSS, and JavaScript fundamentals

## Learning Path

### HTML & CSS Exercises

Practice building static web pages and styling them:

- [Person Profile Page](../html-exercises/person-profile/README.md)
- [Navigation Menu](../html-exercises/navigation/README.md)

- [Recipe Page](../html-exercises/recipe-page/README.md)

**Recommendation**: Complete 3-4 exercises before moving to JavaScript.

### JavaScript Exercises

- [Temperature Converter](../javascript-exercises/temperature-converter/README.md)

- [Image Slider](../javascript-exercises/image-slider/README.md)
- [News Feed](../javascript-exercises/news-feed/README.md)

**Recommendation**: Complete 3-4 exercises before starting Django.

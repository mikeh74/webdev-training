# News Feed

## Exercise overview

In this exercise, you'll practice working with the Fetch API to retrieve and display data from a JSON file.

Your task is to:
1. Use the `fetch()` API to load data from the `feed.json` file
2. Parse the JSON response
3. Display the news articles on the page in a user-friendly format

This exercise will help you understand how to make asynchronous requests and work with external data sources in JavaScript.

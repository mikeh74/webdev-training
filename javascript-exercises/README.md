# JavaScript Exercises

These exercises cover core JavaScript concepts, from variables and functions through to DOM manipulation and fetching data. Work through them to build confidence before starting the main Django project. Aim to complete at least 3 or 4.

## Exercises

1. **[Variable Practice](./variable-practice/README.md)**
   Create variables to store your name, age, and favourite colour.

2. **[Temperature Converter](./temperature-converter/README.md)**
   Build a function that converts Celsius to Fahrenheit and vice versa.

3. **[To-Do List](./todo-list/README.md)**
   Build a simple to-do list where users can add, remove, and mark tasks as complete using DOM manipulation.

4. **[Random Quote Generator](./quote-generator/README.md)**
   Display a random quote from an array when a button is clicked.

5. **[Counter App](./counter-app/README.md)**
   Create increment, decrement, and reset buttons that update a displayed number.

6. **[Image Slider / Carousel](./image-slider/README.md)**
   Create a slideshow that cycles through images with next/previous buttons.

7. **[News Feed](./news-feed/README.md)**
   Fetch and display news articles using a local JSON file.
